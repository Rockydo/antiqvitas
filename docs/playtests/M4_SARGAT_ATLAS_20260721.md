# M4 Sargat atlas batch

Date: 2026-07-21

This thirtieth source-led batch maps the installed Omsk area to a new Sargat
archaeological proxy, resolving 28 previously unmapped controlled locations.
The audited ledger now has 319 selectors, 11,916 locations, and 147 mapped
cultures; the generated catalogue rises to 168 cultures.

The dated Sargat horizon covers the campaign boundary in the Western Siberian
forest-steppe, but the sources describe interaction among Ugrian/Siberian and
southern steppe populations. The row is consequently contested: it is not a
uniform Ugric or steppe identity, language, descent, or political boundary. Its
Uralic/Sami adapter is technical only.

`make validate` passed with all 13,552 populated locations and the
230,000.000-thousand target preserved. The enabled-mod `make smoke` reached
the rendered menu, held it through the post-load check, and passed with zero
new normalized `error.log` lines. The global M4 density target remains open.
