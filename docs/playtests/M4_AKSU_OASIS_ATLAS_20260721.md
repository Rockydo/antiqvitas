# M4 Aksu Oasis atlas batch

Date: 2026-07-21

The forty-first source-led M4 atlas batch assigns the exact installed Aksu
anchor to a new `antq_aksu_oasis` culture. This is a contested oasis proxy, not
a claim about a language, ethnicity, state boundary, or the wider northern
Tarim Basin.

The cited source identifies Aksu as a northwest-Tarim oasis and an important
stop on the northern Silk Road during Han rule from the second century BCE to
the second century CE. The selector is consequently only the campaign-boundary
oasis anchor, and the established Iranian naming adapter is technical rendering
only.

The resulting ledger has 332 selectors, 12,034 resolved locations, 155 mapped
cultures, and a 176-culture M4 definition catalogue.

reported zero new `error.log` lines against the accepted baseline.
