# M4 Vyatka Permic atlas batch

Date: 2026-07-21

This thirty-fourth source-led batch maps the installed Vyatka area to a new
Permic proxy, resolving 18 previously unmapped controlled locations. The
audited ledger now has 323 selectors, 11,989 locations, and 149 mapped
cultures; the generated catalogue rises to 170 cultures.

The source reconstructs an ancestral Permic context across the campaign
boundary, not an exact ethnographic map. The row is therefore contested: it
does not assign a later Komi or Udmurt identity, uniform language,
institutional continuity, or a fixed territorial boundary. Its Uralic/Sami
naming adapter is technical only.

`make validate` passed with all 13,552 populated locations and the
230,000.000-thousand target preserved. The enabled-mod `make smoke` reached
the rendered menu, held it through the post-load check, and passed with zero
new normalized `error.log` lines. The global M4 density target remains open.
