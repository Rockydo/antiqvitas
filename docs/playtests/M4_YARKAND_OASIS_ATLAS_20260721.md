# M4 Yarkand Oasis atlas batch

Date: 2026-07-21

The fortieth source-led M4 atlas batch assigns the exact installed Yarkand
anchor to a new `antq_yarkand_oasis` culture. This is a contested oasis proxy,
not a claim about a language, ethnicity, state boundary, or the wider southern
Tarim Basin.

The *Hanshu* records the territory in the first century BCE as Shache. The
cited source treats a relation of that name to Saka as possible rather than
settled. The selector is consequently only the campaign-boundary oasis anchor,
and the established Iranian naming adapter is technical rendering only.

The resulting ledger has 331 selectors, 12,033 resolved locations, 154 mapped
cultures, and a 175-culture M4 definition catalogue.

Verification: `make validate` passed. `make smoke` launched the enabled mod and
reported zero new `error.log` lines against the accepted baseline.
