# M4 Kucha Oasis atlas batch

Date: 2026-07-21

The thirty-eighth source-led M4 atlas batch assigns the exact installed Kucha
anchor to a new `antq_kucha_oasis` culture. This is a contested local oasis
proxy, not a claim about a language, ethnicity, state boundary, or the wider
northern Tarim Basin.

The *Hanshu* attests Han-era Kucha before AD 1 and the cited reference records
its early-first-century political history. It also states that local language
reconstruction is unsafe for that period. The selector is therefore only the
exact oasis anchor and uses the established Iranian naming adapter as a
technical engine contract.

The resulting ledger has 329 selectors, 12,031 resolved locations, 152 mapped
cultures, and a 173-culture M4 definition catalogue.

Verification: `make validate` passed. `make smoke` launched the enabled mod and
reported zero new `error.log` lines against the accepted baseline.
