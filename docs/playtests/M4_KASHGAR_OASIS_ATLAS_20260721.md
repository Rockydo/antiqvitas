# M4 Kashgar Oasis atlas batch

Date: 2026-07-21

The forty-second source-led M4 atlas batch assigns the exact installed Kashgar
anchor to a new `antq_kashgar_oasis` culture. This is a contested oasis proxy,
not a claim about a language, ethnicity, state boundary, or the wider western
Tarim Basin.

The cited source records Kashgar/Shule in the *Hanshu* and its formal Han
relationship from 59 BCE to 23 CE. The selector is consequently only the
campaign-boundary oasis anchor, and the established Iranian naming adapter is
technical rendering only.

The resulting ledger has 333 selectors, 12,035 resolved locations, 156 mapped
cultures, and a 177-culture M4 definition catalogue.
