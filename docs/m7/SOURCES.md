# M7 Sources and scope

`P14` is the project design bible's military specification in
`ANTIQVITAS_MASTER_PLAN.md` section 14. `CAH-XI` and `BHR` are the source
identifiers defined in `docs/world_1ad/SOURCES.md`. They support the broad
military and Han-frontier contexts respectively; this milestone does not claim
that a particular starting army row is a reconstructed order of battle.

`NOTITIA` refers to the *Notitia Dignitatum* source route named in the plan
appendix. It is used only for the later-age comitatenses/limitanei distinction.
Those types are deliberately unavailable before the Dominate age.

`PER` is the *Periplus of the Erythraean Sea* source route named in the plan
appendix and is used for the Indian-Ocean dhow context. Its use here does not
license trans-oceanic discovery.
