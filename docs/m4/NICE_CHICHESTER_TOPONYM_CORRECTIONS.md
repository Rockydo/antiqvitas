# Nice and Chichester direct AD 1 toponym corrections

The Tier-2 proximity layer previously selected Cemenelum for the installed
Nice field. Pleiades also records the distinct coastal settlement **Nicaea**
at the direct Nice point (`PLE:157894`), so the curated direct layer now takes
precedence. This is a name correction only: it makes no claim about civic
boundary, colony status, population, language uniformity, or the relationship
between the two nearby ancient places.

Pleiades records **Noviomagus** at the direct installed Chichester field
(`PLE:79622`). The direct layer retains that attested name while omitting the
later/expanded tribal qualifier, which the checked record does not require for
the map label. It does not reconstruct the settlement's legal status, extent,
population, road network, or political control.

Both records are declared active across AD 1 in the pinned Pleiades snapshot;
the runtime names are rendered through the existing culture/dialect dynamic
name adapter and source ledger.
