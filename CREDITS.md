# Credits

## Project art

The ANTIQVITAS thumbnail and future original 2D assets are generated for this
project through OpenAI's built-in image-generation pipeline, then inspected and
converted locally. The project reuses Europa Universalis V assets only where
permitted for a game mod and does not redistribute audio.
